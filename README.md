# MF-Angular
Репозиторий для лабораторных работ

Как проходит работа:
1. Выкачиваете папку с лабораторной работой.
2. Создаете новый branch или fork
3. Создаете там проект со своей фамилией на английском.
4. После окончания работы делаете pull request
5. Как только pull request approved и merged - лабораторная работа засчитана
